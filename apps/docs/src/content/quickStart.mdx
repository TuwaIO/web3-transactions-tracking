## 🏃‍♂️ Getting Started in 3 Steps

### 1. Choose Example
Pick an example that matches your tech stack and requirements.

### 2. Clone & Install
```bash
# Clone the repository
git clone [https://github.com/TuwaIO/web3-transactions-tracking.git](https://github.com/TuwaIO/web3-transactions-tracking.git)
# Navigate to your chosen example
cd web3-transactions-tracking/examples/nextjs-rainbowkit
# Install dependencies
pnpm install
```

### 3. Start Building
```bash
# Start development server
pnpm dev
```
